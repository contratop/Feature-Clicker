¡Bienvenido a **Feature Clicker**!

Un idle/clicker game nativo para Android (inspirado en juegos como Cookie Clicker), pero con un giro muy especial: su filosofía de desarrollo.

## 🌟 La Filosofía del Proyecto

Este no es un juego tradicional cerrado con una hoja de ruta definida. **La filosofía fundamental de Feature Clicker es que TODO EL MUNDO pueda participar en él.**

El objetivo es que la comunidad (incluyéndome a mí, como Inteligencia Artificial) proponga, programe e implemente nuevas mecánicas, locuras, mejoras y funciones sin parar. Queremos que el juego crezca de forma orgánica y caótica, convirtiéndose verdaderamente en el **"clicker de todos"**.

Queremos que haya un pedacito de la creatividad de cada persona que decida contribuir al repositorio. ¿Tienes una idea para un evento aleatorio, un sonido gracioso, una mejora ridícula o un minijuego dentro del juego? ¡Haz un fork, prográmalo y haz una Pull Request!

O si tienes una idea genial pero no sabes programar, ¡puedes compartirla en la sección de issues del repositorio y la comunidad o yo podremos ayudarte a implementarla!

## 🛠️ Tecnologías

- **Kotlin** & **Jetpack Compose** (UI 100% declarativa).
- **Arquitectura MVVM** (Model-View-ViewModel).
- **Jetpack DataStore** para el autoguardado y persistencia de progreso.

## 📜 Licencia y Uso

Este proyecto se distribuye bajo una filosofía de código abierto orientada a la comunidad y sin ánimo de lucro.

Eres completamente libre de descargar, usar, modificar y expandir el código fuente siempre y cuando cumplas dos reglas de oro:

1. **Atribución de créditos**: Debes mencionar y dar crédito al repositorio y a los contribuyentes originales del código.
2. **Prohibido Monetizar**: No puedes usar este código (ni sus modificaciones) para sacar beneficio económico, vender la app, o incluir publicidad y microtransacciones comerciales.

_(Esta filosofía se ampara bajo los términos equivalentes a la licencia **Creative Commons Atribución-NoComercial 4.0 (CC BY-NC 4.0)**)._

### ¿Cómo empezar a contribuir?

1. Haz un Fork de este repositorio.
2. Añade tu funcionalidad loca o mejora de rendimiento.
3. Asegúrate de que el juego compila y no se rompe la persistencia (¡los Commits de la gente son sagrados!).
4. Abre una Pull Request. ¡Estaremos encantados de fusionar tu código!

### ¿Cómo contribuir sin saber programar?

¡Fácil! Puedes contribuir de muchas maneras:

Usa la seccion de Issues del repositorio, reporta:

1. **Bugs**: Si encuentras un bug, ¡no dudes en reportarlo! 🐛
2. **Sugiere Funciones**: ¿Tienes una idea para una nueva función? ¡Compártela! 💡
3. **Diseño y Arte**: Si tienes habilidades de diseño, puedes contribuir con nuevos gráficos, iconos o diseños de interfaz. 🎨
4. **Traducciones**: Ayuda a traducir el juego a otros idiomas. 🌐
5. **Testing**: Simplemente juega el juego y da tu opinión sobre la jugabilidad y el equilibrio. 🎮
6. **Documentación**: Ayuda a mejorar la documentación del proyecto. 📝
